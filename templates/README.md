# Templates

This folder becomes the raw material warehouse for EEIK.



Templates contain:

Documents
Code Skeletons
Design Artifacts
Governance Artifacts
Project Assets

The Repository Generator assembles templates into generated repositories.


## Structure

```
templates/
│
├── architecture/
├── documentation/
├── governance/
├── delivery/
├── modernization/
├── ai-engineering/
├── java/
├── aws/
├── react/
├── angular/
├── python/
├── data-engineering/
├── machine-learning/
├── enterprise-architecture/
├── operations/
└── project/
```

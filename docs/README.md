# EEIK Documentation

Welcome to the Enterprise Engineering Intelligence Kit (EEIK).

EEIK is an Enterprise Engineering Operating System designed to accelerate project initialization, engineering governance, architecture design, modernization programs, AI engineering, and enterprise delivery.

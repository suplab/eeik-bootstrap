# Glossary

## Bootstrap Engine

Responsible for project discovery and manifest generation.

---

## Manifest

Source of truth describing project requirements.

---

## Capability Pack

Reusable engineering intelligence unit.

---

## Agent Factory

Generates project-specific agents.

---

## Repository Generator

Creates project workspaces.

---

## Governance Engine

Applies governance controls.

---

## Knowledge System

Captures reusable organizational intelligence.

---

## Workflow

Multi-step engineering process.

---

## Command

User-facing entry point that executes a workflow.

# Knowledge Lifecycle

## Purpose

Knowledge is the most valuable long-term asset within EEIK.

Projects come and go.

Technologies evolve.

Knowledge compounds.

# Phase 1 — Capture

Capture knowledge as close to creation as possible.

Examples:

```text
ADR Created

Incident Closed

Review Completed
```

---

# Phase 2 — Classification

Recommended Categories:

```text
Architecture

Delivery

Operations

Governance

AI

Modernization
```

---

# Phase 3 — Review

Validate:

- Accuracy
- Relevance
- Completeness

---

# Phase 4 — Curation

Remove:

```text
Duplicates

Outdated Information

Low-Value Content
```

Goal:

Maintain quality.

---

```text
knowledge/
```

```text
Lessons Learned
```
